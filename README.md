A collection of 120 programming problems solved as part of summer vacation self-learning assignments.

The repository covers:
- Basic Programming
- Number Theory
- Recursion
- Pattern Printing
- Arrays
- Sorting & Searching
- Strings
- Matrix Operations
- Mini Projects

These problems are commonly asked in companies like TCS, Infosys, Wipro, Amazon, Microsoft, Deloitte, Capgemini, Adobe, and Accenture.
